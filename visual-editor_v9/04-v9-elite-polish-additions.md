# v9 Elite Polish Additions

After the display-text and dirty-state work, use the remaining budget on a few high-value polish upgrades.

Do not sprawl.

## 1. Section Outline Quality

The structure rail is improved.

Finish it with small touches:

- stronger hierarchy between title and type
- better truncation behavior for long titles
- selected-section prominence without noise

## 2. Page Chooser Quality

The page chooser now exists.

Polish it:

- keyboard-friendly search/selection
- clearer current-page indication
- maintain quick routing into visual mode

## 3. Selected-Section Context

Add a more premium sense of context when a section is selected.

Candidates:

- section title chip above inspector
- quick anchor/key display
- compact action row for section-level tasks

This should feel editorial, not developer-centric.

## 4. Small Keyboard Quality

If low-cost:

- improve enter/escape behavior consistency for text overlays
- ensure tab behavior is predictable
- keep shortcuts from conflicting with normal typing

## 5. Regression discipline

No polish change should re-break:

- dirty-state semantics
- overlay editing
- section switching
- viewport switching

## Non-goals

- no new inspector rearchitecture
- no new drag/drop system
- no new custom-section project
